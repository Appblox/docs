Blox Store Walkthrough

Blox Store​

The Blox store is a registry where developers can create and publish Blox or download Blox published by other developers.

Search​

You can search for different Blox that cater to your needs by entering keywords in the search bar, as shown in the above picture.

Filter​

You can filter the store for Blox by the following conditions: Blox Type, Language/Technology, Category

Sort​

Sort the Blox store by different parameters by clicking the sort button on the top-right.

Visiting a Blox​

When you click any Blox in the Blox store, you are redirected to the ___ page. It gives you all details about the particular Blox. There are four tabs present in it, viz. ReadMe, Dependencies, Versions and Composable Blox. If the Blox you are visiting is an App Blox, there will be a 5th tab called Structure.

ReadMe​

The ReadMe tab contains general information about this particular blox. To reuse a blox in your app, copy the link given below "Install" on the top right corner of this tab and ___.

ReadMe also gives the number of downloads, tags, list of contributors and other such useful information about the blox.

Dependencies​

In the Dependencies tab, as shown below, all dependent bloxes of the chosen blox is listed down. Click here to read more.

Versions​

The Versions tab will display all versions of the blox till date.

Composible Blox​

All composible bloxes are listed in this tab as shown below. Click here to read more.

Structure​

If the visited blox is an App Blox, the Structure tab will be present. It will display the entire app structure.

As shown below, the flow diagrams give you a good understanding of how the UI bloxes are connected to function bloxes and the function interfaces present in the app blox.

Another salient feature is that testing can be performed using the test endpoints.
